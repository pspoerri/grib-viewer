# 05 — Variable catalog, quantization, ladders, colormaps

One central table (single source of truth, no per-model overrides)
drives wire quantization, default colormaps, display ranges, temporal
classification, and reducer kinds. Unknown GRIB parameters get a
generated id and generic defaults so arbitrary debug output still
renders.

## Wire quantization

int16, `value = raw·scale + offset`, **−32768 = NoData**, finite
clamped to ±32767. VMin/VMax are colormap hints only.

Comfort diagnostics (wetbulb_2m, heat_index, theta_e, dewpoint
depression): temperature quantization, `temp` colormap, no ensemble
products.

**Reducer kinds** feed the client-side daily summary and the
window-op defaults: minmax → {min,max,+times}, max (gusts/wind), mean
(flux/percent), sum (accumulation), period (probabilities: temporal-OR
across a day / peak on maps).

## Isobaric levels

Standard set **850 / 700 / 500 / 300 hPa** (+250 where upstream has
`fi`). Indicative ranges for quantization offsets: fi_300 ≈ 8400–9900,
fi_500 ≈ 4700–6000, fi_850 ≈ 1100–1700 gpm; t_500 ≈ 230–280 K; wind
@300 up to 110 m/s.

## Threshold ladders (curated exceedance defaults)

- Precip rate: **0.1 / 1 / 2 / 5 / 10 mm/h**
- Gusts (vmax_10m): **Bft 5/7/8/10/12 = 8 / 14 / 17 / 25 / 33 m/s**
- Temperature: frost (≤0 °C), ≥25 °C, ≥30 °C
- Global radiation: **120 / 400 / 800 W/m²**
- Beaufort table (m/s): 0, 0.3, 1.6, 3.4, 5.5, 8, 10.8, 14, 17,
  20.8, 25, 28.5, 33

All ladders are UI conveniences over the same arbitrary-threshold
engine — nothing is precomputed.

## Ensemble products

Percentile UI set **[10, 25, 50, 75, 90]** (+ p0/p100 as Min/Max),
arbitrary `_p{N}` accepted by the API. Bare id = p50. `_mean`,
`_ctrl`, `_spread` (p90−p10), `_m{N}` raw member.

but not `pmsl`, lacks td_2m/cloud-split; iconch1 sources hsurf from
vertical_constants; DWD EPS files may carry members 1..N with no
perturbation 0 (control = lowest number).

## Colormaps

Registry served by `/api/colormaps` as RGBA stops (position 0..1,
per-stop alpha). Set: viridis, plasma, inferno, magma, greys, blues,
greens, coolwarm, bwr, seismic, **temp** (stepped-capable), **wind**,
**precip** (log-scaled, alpha 0 at 0), **cloud** (alpha ramp),
**relhum**, **solar**, **snow**, **prob** (transparent → pale yellow →
orange → magenta → deep purple; 0 % reads through to the basemap).
Log color mapping applies to precip palettes (floor vmin if >0 else
vmax/1000). Temperature stepping bands: 2 °C < −30, 1 °C in
[−30,40], 2 °C in [40,50], 5 °C above — baked into ramp textures and
legend alike.
